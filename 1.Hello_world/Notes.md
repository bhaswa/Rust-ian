##Installation:
	curl --proto '=https' --tlsv1.3 https://sh.rustup.rs -sSf | sh

##Version:
	rustc --version

##Updating
	rustup update
	
##Uninstalling
	rustup self uninstall
	
##Compilation:
	rustc main.rs
	
##Build cargo Package
	cargo new <PACKAGE_NAME>
	
##Build Program
	cargo build
	
##Run Program
	cargo run
	
##Check Program for errors
	cargo check
